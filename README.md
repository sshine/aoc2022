# Advent of Code 2022

Advent of Code 2022 can be found on this address: https://adventofcode.com/2022

## Directory structure

Put your files in a directory corresponding to your first name in lowercase:

```
adventofcode
└── aoc2022
    ├── README.md
    ├── simon
    ├── <name>
    ├── ...
    └── inputs
        ├── day01.txt
        ├── day02.txt
        ├── ...
        └── day25.txt
```

As you can see, the input for each day is shared.
